🛍️ Teddy's E-Commerce Website
A fully responsive, modern e-commerce website built with HTML5, CSS3, and vanilla JavaScript. Includes product listing, shopping cart, blog, about, contact page with Formspree integration, and interactive UI components.

📸 Preview
https://img/hero4.png
Homepage hero section

✨ Features
✅ Responsive Design – Works perfectly on desktop, tablet, and mobile

✅ Product Pages – Shop page with grid layout and product cards

✅ Single Product Page – Image gallery, size selector, quantity input

✅ Shopping Cart – Add/remove items, update quantities, apply coupons

✅ Blog Section – Stylish blog layout with date badges

✅ Contact Form – Integrated with Formspree (no backend required)

✅ Google Maps – Embedded map showing store location

✅ WhatsApp Chat Button – Floating button for instant customer support

✅ FAQ Section – Common questions with answers

✅ Newsletter Signup – Email subscription section

✅ Mobile Menu – Hamburger menu with smooth slide-in animation

✅ Coupon System – Apply discount codes (SAVE10, WELCOME20, FREESHIP)
